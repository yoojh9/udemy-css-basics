# HTML-FUNDAMENTALS

## 1) Text Elements

- 각 페이지에 \<h1\>은 하나만 존재해야 한다.
- \<b\> 태그를 써서 글자를 강조하는 것은 권장되지 않는다. b 태그는 레거시 태그로 HTML5에서는 b 대신 시맨틱의미가 있는 \<strong\> 엘리먼트가 대신한다.
- 이탤릭체도 마찬가지로 예전에는 \<i\> 태그를 썼지만 HTML5에서는 emphasis를 의미하는 \<em\> 태그를 사용한다.
